Technology Stack

Flay is built on python flask. All recipe data is imported from Kaggle into Mongodb. The front end was made with Jinja and Twitter Bootstrap.

Orchestration

Docker

This project is configured to create a ubuntu image for the flask app with a shared /app folder for development and a mongodb image.

Vagrant

This project is also configured to create a vagrant image with the same config as docker, in case you'd rather use a VMWare provider.

Kubernetes

This project contains yaml files for a kubernetes deployment. Why use three orchestration tools platforms? To show off.
